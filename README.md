# 图片编辑工具

一个简单易用的本地图片编辑网页小工具，所有操作均在浏览器中完成，保护您的隐私。

## 功能特性

### 图片导入
- 单张图片上传
- 批量图片上传（最多50张）
- 拖拽上传支持

### 图片编辑
- **场景模板选择**：
  - 手机壁纸（9:16）
  - 桌面壁纸（16:9）
  - 社交媒体（1:1）
  - 博客封面（1.91:1）

- **尺寸调整**：
  - 自定义宽度和高度
  - 保持比例选项
  - 支持常用尺寸

- **裁剪与变换**：
  - 旋转（左/右）
  - 水平翻转
  - 垂直翻转

### 导出与分享
- 单张图片导出
- 批量导出所有图片
- 图片自动下载

## 使用方法

1. 打开 `index.html` 文件（直接在浏览器中打开即可）
2. 点击上传区域或拖拽图片到上传区域
3. 选择要编辑的图片（点击图片列表中的图片）
4. 使用各种编辑功能：
   - 选择场景模板
   - 调整尺寸
   - 旋转或翻转图片
5. 点击导出按钮保存编辑后的图片

## 技术特点

- **纯前端实现**：无需服务器，所有操作在浏览器中完成
- **响应式设计**：支持桌面和移动设备
- **隐私保护**：图片不会上传到服务器
- **简单易用**：直观的界面，无需学习成本

## 系统要求

- 现代浏览器（Chrome、Firefox、Safari、Edge）
- 支持HTML5 Canvas
- 建议使用最新版本的浏览器以获得最佳体验

- 最多支持上传50张图片
- 导出的图片格式为PNG
- 编辑过程中建议不要刷新页面，以免丢失进度
